# **App Name**: Planet - Cartera

## Core Features:

- Admin Authentication: Admin login with username 'Cristobal' and password '0120'.
- Dashboard Overview: Dashboard displaying key metrics like 'Total Debt', 'Total Clients', 'Recovered Clients', and 'Recovery Rate'.
- Plaza-Specific Panels: Individual panels for each Plaza (AUTLAN PREPA, CREDIMEX, UNION DE TULA, TECOLOTLAN, OFICINA CENTRO, RUTA AARON) showing 'Pending Debt' and 'Recovery Rate'.
- Excel Import: Functionality to import client data from Excel files with fields: PLAZA, FECHA, NOMBRE, DIRECCION, TELEFONO, AVAL, TELEFONO, PRESTAMO, PAGO, NO.VENC., ADEUDO.
- Data management per plaza: Each Plaza has its panel where the client's data is stored.
- Range Debt Selector: Displays all debts based on a specified range or number of due payments

## Style Guidelines:

- Primary color: Midnight Blue (#191970) for a professional and reliable feel.
- Background color: Light Gray (#F0F0F0) for a clean and modern look.
- Accent color: Coral (#FF7F50) to highlight key metrics and interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use simple and clear icons to represent different actions and data categories.
- Dashboard layout with clear sections for overall metrics and individual plaza performance. Inspired by the provided screenshot for consistency.
